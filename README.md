# 🔒 NoteVault

An immersive, visual-heavy web application designed for seamlessly taking, editing, and organizing notes right in your browser. NoteVault avoids flat minimalism in favor of rich layout design, card effects, glowing gradients, and full client-side security structure.

## 🚀 Key Features

* ⚡ **Persistent Storage:** Instant saving using the client's LocalStorage.
* 🔍 **Instant search bar:** Dynamic UI filtration as you type.
* ✏️ **Modal Editors:** Rich popover modal workspaces to construct or revise entries.
* 🎨 **Non-Minimalist Aesthetic:** Styled with modern gradients, depth elements, and glowing outlines.

## 🛠️ How to Launch Locally

1. Download the files to a single directory.
2. Direct-launch **`index.html`** in any web browser.
3. Enjoy your local vault!
